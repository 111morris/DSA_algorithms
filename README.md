## Data Structure and Algorithms

Data structure *is a way to organize and store data efficiently (in terms of performance and memory)*

Algorithms *they are set of instructions 

### TYPES OF DATA STRUCTURES

### Primitive Data Structures

Basic building blocks.

* Integer
* Float
* Character
* Boolean
* Pointer

### **Non-Primitive (Composite) Data Structures**

#### **Linear Data Structures**

Data stored sequentially.

* **Array**
* **String**
* **Linked List**
  * Singly
  * Doubly
  * Circular
* **Stack** (LIFO)
* **Queue** (FIFO)
  * Simple Queue
  * Circular Queue
  * Priority Queue
  * Deque

#### **Non-Linear Data Structures**

Data stored hierarchically or graph-like.

* **Tree**
  * Binary Tree
  * Binary Search Tree (BST)
  * AVL Tree
  * Red-Black Tree
  * Heap
  * Trie
  * Segment Tree
* **Graph**
  * Directed / Undirected
  * Weighted / Unweighted

### **Hash-Based**

* **Hash Table**
* **Hash Map**
* **Hash Set**
