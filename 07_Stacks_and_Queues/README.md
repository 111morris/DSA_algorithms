# Stacks and Queues

## Theory

### Stack
LIFO (Last In First Out).
*   **Operations**: Push, Pop, Peek.

### Queue
FIFO (First In First Out).
*   **Operations**: Enqueue, Dequeue.

### Circular Queue
The last position is connected back to the first position to make a circle.
